# 💰 Smart Expense Splitter

A real-time collaborative expense management platform built with the MERN stack.

## 🚀 Live Demo
- Frontend: [https://smart-expense-splitter-newn.vercel.app/]
- Backend API: [https://smart-expense-splitter-58qq.onrender.com]

## ✨ Features
- Multi-method expense splitting (Equal, Exact, Percentage, Shares)
- Debt simplification algorithm (minimizes transactions)
- Real-time activity notifications
- PDF/CSV export
- Interactive analytics dashboard

## 🛠 Tech Stack
- **Frontend:** React, Tailwind CSS, Framer Motion, Recharts
- **Backend:** Node.js, Express, MongoDB, Socket.io
- **Database:** MongoDB Atlas

## 🔧 Local Setup

```bash
# Clone
git clone https://github.com/YOUR_USERNAME/smart-expense-splitter.git
cd smart-expense-splitter

# Backend
cd backend
npm install
# Create .env file (see below)
npm run dev

# Frontend (new terminal)
cd frontend
npm install --legacy-peer-deps
npm run dev
